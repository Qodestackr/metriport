---
title: "Limits and throttling"
description: "Information about Metriport's limits related to request size and request frequency"
---

The following table outlines the current limits for certain operations on the Metriport platform.

If your operation usage pattern needs higher limits, please reach out to `support@metriport.com`.

| Operation                     | Max requests per minute | Request rate      |
| ----------------------------- | :---------------------: | :---------------: |
| Patient Create or Update      | 15                      | 1 every 4 seconds | 
| Document Query Start          | 20                      | 1 every 3 seconds |
| Consolidated Data Query Start | 120                     | 2 every 1 second  |
